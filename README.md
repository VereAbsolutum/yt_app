# YT Projeto Yii2

Obs.: Certifique-se de estar usando o PHP 7.2 ou superior. 

## Instalação do Yii2
Para instalar o Yii2, siga os seguintes passos:

1. Certifique-se de que você tenha o Composer instalado em seu sistema. Se você não tiver, pode instalá-lo seguindo as instruções em getcomposer.org.
2. Abra um terminal ou prompt de comando e navegue até o diretório onde deseja criar o projeto.
3. Execute o seguinte comando para criar um novo projeto Yii2 usando o modelo de projeto básico:


Isso criará um novo diretório chamado basic contendo o código do projeto.

## Executando o Projeto
Para executar o projeto, siga os seguintes passos:

Abra um terminal ou prompt de comando e navegue até o diretório do projeto.
Inicie o servidor de desenvolvimento interno do PHP executando o seguinte comando:
php yii serve

Isso iniciará o servidor e você poderá acessar o aplicativo em http://localhost:8080.

## Instalando o Servidor PHP XAMPP no Windows
Para instalar o servidor PHP XAMPP no Windows, siga os seguintes passos:

1. Acesse a página de download do XAMPP em apachefriends.org.
2. Clique no botão de download para a versão do Windows e salve o arquivo em seu computador.
3. Execute o instalador e siga as instruções na tela para instalar o XAMPP.
4. Depois de concluir a instalação, inicie o painel de controle do XAMPP e inicie os serviços Apache e MySQL.

Agora você pode usar o servidor PHP XAMPP para executar seus projetos PHP no Windows.
